Déclaration d'accessibilité

Nous nous engageons à rendre notre site web accessible à tous, y compris aux personnes en situation de handicap. Cette déclaration d'accessibilité décrit les mesures que nous avons prises pour améliorer l'accessibilité de notre site.

Conformité aux normes

Notre site web est conçu pour être conforme aux directives pour l'accessibilité des contenus Web (WCAG) 2.1 niveau AA. Nous avons effectué des tests pour nous assurer que notre site est accessible et utilisable par tous.

Fonctionnalités d'accessibilité

  • Navigation au clavier : Vous pouvez naviguer sur notre site en utilisant uniquement le clavier.
  • Contraste élevé : Un mode de contraste élevé est disponible pour améliorer la lisibilité.
  • Descriptions alternatives : Toutes les images ont des descriptions alternatives pour les lecteurs d'écran.

Retour d'information

Si vous rencontrez des difficultés pour accéder à notre site ou si vous avez des suggestions pour améliorer l'accessibilité, veuillez nous contacter à l'adresse suivante : contact@coursedelezards.fr.

À propos

La Course de Lézards est le premier site français dédié aux courses de lézards professionnelles.
